# PSConfigMaster

PowerShell консольный менеджер для управления всеми настройками PowerShell.

## Возможности

- Управление профилями PowerShell
- Установка/удаление модулей
- Настройка Execution Policy
- Управление цветами консоли
- Работа с переменными окружения
- Создание/удаление псевдонимов
- Управление функциями
- Настройка PSReadLine
- Сброс всех настроек

## Запуск

```powershell
.\PSConfigMaster.ps1
